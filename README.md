Build a polished, ATS-safe resume in minutes — no account, no data sent anywhere, everything runs in your browser.

## Features

- **Two clean templates** — Classic and Minimal, both designed for hiring managers
- **Section reorder** — drag sections into any order directly from the sidebar
- **Accent colour picker** — personalise with five curated presets or a custom colour
- **Smart pagination** — automatically computes page breaks for multi-page resumes
- **One-click PDF** — pixel-perfect export via the browser print API
- **Fully browser-based** — no backend, no account, no tracking

## Tech stack

| Layer | Tech |
|---|---|
| Framework | Next.js 15 (App Router) |
| UI | React 19, Tailwind CSS v4 |
| State | Zustand 5 |
| Drag & drop | dnd-kit |
| PDF export | Browser print API |
| Deploy | Vercel |
